\begin{abstract} 
  This meta-paper present a linearization to the existing formulation of the maximally diverse grouping problem. Computational experiments performed on a series of random graphs as well as on several instances of the maximum diversity problem taken from the literature, shows that most of the time, the linearization is faster than the quadratic formulation, using the Simplex method to solve them.
\end{abstract}

\begin{resumo} 
  Este meta-artigo apresenta uma linearização para uma formulação existente do problema da máxima diversidade de grupos. Experimentos computacionais realizados utilizando \textit{IBM ILOG CPLEX} em uma série de grafos aleatórios, bem como em várias instâncias do problema da máxima diversidade de grupos retiradas da literatura, mostram que grande parte das vezes a linearização tem melhor desempenho.
\end{resumo}

\section{Introdução}

O Problema da Diversidade Máxima de Grupos (PDMG), da área de Otimização Combinatória, consiste em encontrar uma maneira de particionar um conjunto de \textit{n} elementos em \textit{m} subconjuntos menores, chamados grupos, de modo que a diversidade entre os elementos em cada grupo seja a maior possível. A diversidade entre os elementos de um grupo é calculada como a soma da distância individual entre cada par de elementos. O objetivo do problema é maximizar a diversidade geral, i.e., a soma da diversidade de todos os grupos.

Em \cite{feo:11} foi provado que o PDMG é NP-difícil. De maneira geral, na prática, os problemas de otimização desta natureza são caracterizados por grandes espaços de soluções dificultando, com isso, a obtenção de uma solução de custo ótimo em tempo computacional razoável.

O PDMG também é conhecido como problema de k-partições(\textit{k-partition Problem} \cite{feo:22}), ou Problema de Partição Equilibrada (\textit{Equitable Partition Problem} \cite{mingers:22}). Além disso o Problema da Diversidade Máxima (\textit{Maximum Diversity Problem}, \cite{kuo:11}) é o caso mais explorado da literatura em que o número de grupos é sempre igual a 1.

Várias são as aplicações do PMDG, como exemplo nós temos projetos de circuitos VLSI (\cite{chen:11}); Armazenamento de memória paginada (Kral, 1965). Uma das mais populares aplicações do PDMG aparece no contexto acadêmico, quando formamos grupos de estudantes \cite{mingers:11}, a situação também pode ser adaptada para a criação de grupos de revisores para publicações científicas ou para avaliação de projetos em agências de fomento à ciência \cite{hettich:11}. Criando diversos grupos onde pessoas com conhecimentos e experiências diferentes lidem com a heterogeneidade e facilite a comunicação entre elas (Bhadury et. al [2]).\cite{bhadury:11}. 

Neste trabalho é apresentada uma formulação linearizada do modelo quadrático existente na literatura, juntamente com o comparativo entre o desempenho das formulações.


\section{Formulações matemáticas}
Existem duas variantes do PDMG. A mais conhecida é (PDMG1), onde todos os grupos são forçados a ter o mesmo número $k$ de elementos, tal que $k = n / m$. A segunda variação, (PDMG2) permite que os grupos tenham tamanhos variados. 

Assim, o número $c_g$ de elementos em um grupo é dado pelo intervalo de $[a_g, b_g]$, onde $a_g \leq b_g$ para $g = 1, 2, ..., m$. Ambas as variantes podem ser modeladas como um problema de programação inteira. Considerando-se $x_{ig} (i = 1, 2, ..., n e g = 1, 2, ..., m)$ as variáveis binárias definidas de maneira que:

{
$$\mbox{$x_{ig}$} = 
	\left \{ 
		\begin{array}{ll}
			1,\ $Se o elemento \textit{i} pertence ao grupo \textit{g}$ \ \\
			0,\ $Caso contrário$
		\end{array}
	\right. $$
}

\subsection{Formulação quadrática}

\begin{equation}
	\max \quad \sum_{g=1}^{m} \sum_{i=1}^{n} \sum_{j = 1}^{n} d_{ij}x_{ig}x_{jg} \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \
\end{equation}
\textrm{Tal que \newline}
\begin{equation}
	\quad \sum_{g=1}^{m} x_{ig} = 1, \ \ i = 1, 2, \ldots, n, \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \
\end{equation} 
\begin{equation}
	\quad \sum_{i=1}^{n} x_{ig} \geq  a_{g}, \ \ g = 1, 2, \ldots, m, \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \
\end{equation}  
\begin{equation}
       \quad \sum_{i=1}^{n} x_{ig} \leq  b_{g}, \ \ g = 1, 2, \ldots, m \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \
\end{equation}
\setcounter{equation}{0}
Restrição (2) exige que cada elemento $i$ esteja em apenas um grupo. Restrições (3) e (4) garantem que cada grupo $g$ contenha ao menos $a_g$ e não mais que $b_g$.
% Segunda formulação

A linearização do modelo consiste na adição de uma variável que substitue o fator quadrático. Considerando-se $y_{ijg} (i = 1, 2, ..., n, j = 1, 2, ..., n e g = 1, 2, ..., m)$ as variáveis binárias definidas de maneira que:
{
$$\mbox{$y_{ijg}$} = 
	\left \{ 
		\begin{array}{ll}
			1,\ $Se os elementos \textit{i} e \textit{j} pertencerem ao grupo \textit{g}$ \ \\
			0,\ $Caso contrário$
		\end{array}
	\right. $$
}
\subsection{Formulação linear}

\begin{equation}
	\max \quad \sum_{g=1}^{m} \sum_{i=1}^{n} \sum_{j = 1}^{n} d_{ij}y_{ijg}  \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \
\end{equation}
\textrm{Tal que \newline}
\begin{equation}
	\quad \sum_{g=1}^{m} x_{ig} = 1, \ \ i = 1, 2, \ldots, n, \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \
\end{equation} 
\begin{equation}
	\quad \sum_{i=1}^{n} x_{ig} \geq  a_{g}, \ \ g = 1, 2, \ldots, m, \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \
\end{equation}  
\begin{equation}
       \quad \sum_{i=1}^{n} x_{ig} \leq  b_{g}, \ \ g = 1, 2, \ldots, m \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \
\end{equation}
\begin{equation}
	\quad x_{ig} + x_{jg} - 1 \leq y_{ijg}, \ \ g = 1, 2, \ldots, m; i\ e \ j = 1, 2, \ldots, n, \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \
\end{equation}  
\begin{equation}
	\quad \sum_{i = 1}^{n} \sum_{j = 1}^{n} y_{ijg} \geq (a_g - 1) x_{jg} \ \ g = 1, 2, \ldots, m, \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ 
\end{equation}  
\begin{equation}
       \quad \sum_{i = i}^{n} \sum_{j = 1}^{n} y_{ijg} \leq (b_g - 1) x_{jg}, \ \ g = 1, 2, \ldots, m, \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \
\end{equation}  
As restrições (2), (3) e (4) da formulação quadrática serão mantidas. A restrição (5) exige que a variável $y_{ijg}$ receba o valor 1 se, e somente se, as variávei $x_{ig}$ e $x_{jg}$ respectivamente forem diferentes de 0. Restrições (6) e (7) garantem que a quantidade de elementos selecionados para o grupo $g$ em relação a variável $y_{ijg}$ seja ao menos $a_g$ e no máximo $b_g$.

\section{Problema da Diversidade Máxima de Grupos}
Diversas abordagens heurísticas e metaheurísticas têm sido propostas para obter soluções de alta qualidade para o PDMG.

Fan et al. (2011) abordaram o PDMG através de um algoritmo híbrido baseado em algoritmos genéticos. Esta abordagem mostrou-se satisfatória encontrando-se bons resultados para instâncias de tamanho máximo n = 240.

Gallego et al. (2013) propuseram uma nova heurística utilizando mecanismos de busca, com base na metodologia busca tabu para o PDMG, incluindo uma estratégia de oscilação, que permitiu o cruzamento de caminhos viáveis de busca. Foram avaliados procedimentos de construção e de busca para configurar um procedimento de solução que quando comparado ao estado da arte para o MDGP. Extensos experimentos computacionais com médias e grandes instâncias foram apresentados para demonstrar as vantagens do método de solução que inclui estratégica de oscilação.

Palubeckis et al. (2011) propuseram três heurísticas para o PDMG, sendo elas: Multistart simulated annealing (MSA), Hybrid steady-state genetic algorithm (HGA) e Variable Neighborhood search (VNS). O desempenho das heurísticas foi comparado computacionalmente através de dois grupos de instâncias: o primeiro, instâncias do Problema da Diversidade Máxima, caso particular do PDMG e o outro, com instâncias de até 2000 elementos. Dentre as três abordagens, a que se mostra mais eficiente é o HGA. 

Uro?evi? (2014) desenvolveu uma heurística híbrida baseada em VNS onde o processo de busca utilizado para melhoria de soluções baseou-se em um algoritmo VND composto por duas vizinhanças. Os resultados apresentados mostram que a metodologia proposta é $0,5\%$ melhor que os resultados obtidos com o VNS clássico.

Rodriguez et al. (2013) apresentam um algoritmo Artificial Bee Colony (ABC) que é uma metodologia baseada no comportamento alimentar das abelhas. A eficiência do novo algoritmo foi comprovada para instâncias de até 2000 elementos e, comparado aos seus antecessores, este algoritmo apresentou as melhores soluções até o presente momento.

Diante da análise feita a partir de trabalhos anteriores, é notório o fato de que não são utilizadas abordagens exatas para solucionar o PDMG, uma vez que algoritmos aproximativos na maioria das vezes não conseguem encontrar a solução ótima mas, são considerávelmente mais rápidos para atingir resultados aceitáveis.

\section{Resultados Computacionais}
Nessa seção serão descritos os experimentos computacionais realizados para testar a eficiência das implementações de ambos os modelos utilizando o solver CPLEX Optimizer e a linguagem de programação C++. Toda computação foi feita em um Intel Core I5-4570 (3.20GHz) 4 threads com 14.6 GB RAM e sistema operacional Linux Mint 18 Cinnamon 64 bits.

Os experimentos foram realizados sobre uma amostra de 240 das 480 instâncias disponibilizadas do problema. Este conjunto pertence á MDGPLIB, pode ser encontrado em http://www.optsicom.es/mdgp e é dividido em três subconjuntos: \\

\textit{RanReal:} 160 instâncias compostas de matrizes $N$ x $N$ em que os valores das instâncias são números reais gerados usando uam distribuição uniforme \textit{U}$(0, 100)$. O número de elementos $n$, o número de grupos $m$ e os limites de cada grupo $a_g$ e $b_g$ são mostrados na 1. São 20 instâncias para cada combinação de parâmetros, 10 instâncias para o caso em que todos os grupos devem ter o mesmo tamanho e 10 instâncias de grupos de tamanhos diferentes. No primeiro caso, $a_g$ = $b_g$ = $n/m$. Já no segundo caso, os limites de cada grupo ($a_g$ e $b_g$) foram gerados aleatoriamente em um intervalo pré-definido. Com isso ${a_g}$ é gerado no intervalo [${a_g}^{min}$, ${a_g}^{max}$] e ${b_g}$ no intervalo [${b_g}^{min}$, ${b_g}^{max}$]. Esse conjunto de instâncias foi introduzido por (Fan et. al [8], 2011) com $n$ de 10 a 240. (Gallego et. al 2013) geraram instâncias maiores, de 480 e 960 elementos.

O número de coordenadas para cada instância é aleatoriamente gerado no intervalo entre 2 e 21. (Glover et. al, 1998) propôs esse gerador para o PDM.

Considerando os resultados obtidos com as intâncias de tamanhos iguais, podemos assumir que, para as que são do tipo \textit{RanReal} e \textit{RanInt} a linearização obteve uma média de resultados consideravelmente maiores ao final da execução. O modelo quadrático, nas instâncias onde $n$ é menor, ou igual a 12 levaram mais tempo para atingir a solução ótima. Já nas instâncias maiores, o GAP da média das soluções é até 34 vezes mais alto que o da modelagem com função objetivo linear.

Já nas instâncias do tipo \textit{Geo} é baixíssima a diferença entre os resultados obtidos, o mesmo vale pro GAP, pois ambas as modelagens ficam próximas de atingir o valor ótimo global.

A maior diferença apresentada nas tabelas 2 e 3 se dá pelo fato da quantidade de nós gerados, em todos os casos a média de nós gerados pelo modelo quadrático é maior que a quantidade de nós gerados pelo modelo linearizado e a média de tempo para as instâncias maiores é menor, assim é possível assumir que solucionar as instâncias utilizando o modelo quadrático demanda bem mais memória.

\section{conclusões e sugestões para trabalhos futuros} 
Neste trabalho foi apresentado um modelo linearizado para o PDMG. Além disso, foi feito um comparativo de desempenho entre ambas as modelagens, levando em comparação diversos fatores, dentre eles o resultado obtido, o GAP, tempo de resolução e a quantidade de nós gerados. Foi mostrado que a linearização na maioria dos casos obtém em média resutados mais próximos do ótimo global, um valor de GAP mais baixo, gera menos nós na árvore e com isso consome menos memória durante a execução.

Como trabalhos futuros propõe-se a implementação de uma relaxação lagrangeana e maneiras de aumentar o desempenho de resolução do modelo.
